# VidSnapsAI 🎬🤖

A Python-based project that generates short video reels using AI.

## 🚀 Features
- Auto-selects images from folders
- Converts text to speech using gTTS
- Uses FFmpeg to generate engaging reels
- Output video saved as `reel.mp4`
